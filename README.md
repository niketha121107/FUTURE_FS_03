# 🌿 Leaf & Bloom

A modern, responsive online plant nursery built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. Leaf & Bloom offers a seamless shopping experience for plant lovers with a clean, elegant interface and intuitive navigation.

---

## 🌱 Features

- 🏡 Modern and responsive landing page
- 🌿 Browse plants by category
- 🔍 Search and filter plants
- 💰 Price range filtering
- 🛒 Product detail cards
- 📱 WhatsApp inquiry integration
- 📖 Dedicated **Our Story** page
- 🎨 Beautiful UI with smooth animations
- ⚡ Fast performance using Vite

---

## 📂 Categories

- Indoor Plants
- Outdoor Plants
- Flowering Plants
- Succulents
- Seeds
- Bulbs

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/leaf-and-bloom.git
```

### Navigate into the project

```bash
cd leaf-and-bloom
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
